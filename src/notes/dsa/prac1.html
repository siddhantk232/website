1. Aim: Insert an element at position n in an array.

    To insert an element at position n:

    • We move each element one index after its current position. Iterating from last to n (inclusive) and moving ith to (i+1)th position is one way to do it.
    • we simply put a desired value (could be a user input) at index n.
    #include <stdio.h>

int insert(int* arr, int size, int pos, int elem) {
    for (int i = size - 1; i >= pos - 1; --i) {
        arr[i + 1] = arr[i];
    }

    arr[pos - 1] = elem;
    return size + 1;
}

int main() {
    int n = 5;
    int arr[7] = {1,2,3,4,5};

    int pos = 3;
    int elem = 999;

    n = insert(arr, n, pos, elem);

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}

    1 2 999 3 4 5

  2. Aim: Delete an element at position n in an array.

    To delete an element at position n:

    • we iterate from postion n till the last and shift every element one index back from it’s original position.
    • The size of array (n) is decremented by 1 after the above operation.
    #include <stdio.h>

int main() {
    int pos = 3;
    int elem;
    int n = 5;
    int arr[7] = {1,2,3,4,5};

    for (int i = pos - 1; i < n - 1; i++) {
        arr[i] = arr[i+1];
    }

    n--;

    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}

    1 2 4 5

  3. Aim: Perform linear search for an element in an array.

    Linear Search is a simple search algorithm. We iterate over all the elements of an array to look for the target element.

    The below program prints the index of the found target or, exits with a failure message if no target element exists. 

    #include <stdio.h>

int main() {
    int arr[] = {1,2,3,99,2389,21};
    int size = sizeof(arr) / sizeof(int);
    int target = 99;

    int found_index = -1;

    for (int i = 0; i < size; ++i) {
        if (arr[i] == target) {
        found_index = i;
        }
    }

    if (found_index >= 0) {
        printf("Element %d found at index %d\n", arr[found_index], found_index);
    } else {
        printf("element not found");
    }

    return 0;
}

    Element 99 found at index 3
