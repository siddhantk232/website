CSS

  • Introduction
  • How to write css for our webpage
    • Inline styles: Use the style HTML attribute to declare basic properties. eg: <h1 style="color: white; background-color: black;">This is a white text with black background</h1>

    • The style tag: Use the <style></style> tag inside your head tag to target and style html elements of your page. eg: 

      <html>
  <head>
    <style>
      h1 {
        color: white;
        background: black;
      }
    </style>
  </head>
  <body>
    <h1>White text on black bg</h1>
  </body>
</html>
